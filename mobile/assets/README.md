# Assets para Azul Mobile

## Archivos requeridos para compilar APK:

### icon.png
- **Tamaño**: 512x512 px
- **Formato**: PNG con transparencia
- **Diseño**: Logo de Azul (esfera azul con gradiente)

### splash.png (opcional)
- **Tamaño**: 1920x1080 px
- **Formato**: PNG
- **Diseño**: Pantalla de carga con branding

## Generar iconos:

Puedes usar cualquier editor de imágenes o herramientas online:
- [Icon Generator](https://romannurik.github.io/AndroidAssetStudio/)
- Photoshop / GIMP
- Canva

## Placeholder:

Si no tienes iconos personalizados, buildozer usará el icono por defecto de Kivy.
